About

This page shows all shared components. It can be used to develop new shared components or to maintain existing ones.

If you'd like to see the projects Sling Academy or Star Wars Playground in action, please consult the README to see how to start them.

Pagination

The component always shows the first and last two pages plus the current one, and it's two previous and next pages. If there are more then 9 pages to be shown in the pagination, dots will be used to reduce the amount of rendered HTML elements (e.g. [1, 2, '...', 15, 16, 17, 18, 19, '...', 99, 100]).

Toggle

The toggle component renders a toggle/switch with a label text on the right to it. Optionally a tooltip text can be provided. The tooltip is only shown, when a text is provided.

Loading modal

The loading modal is a component that must be used with the NgbModal service. The component itself just renders a text and a spinner. Handling the modal must be done via NgbModal.
